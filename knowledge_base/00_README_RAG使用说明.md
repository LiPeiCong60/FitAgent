# FitAgent RAG 知识库使用说明

## 用途
这套文档用于「AI 健身助手 / FitAgent」的 RAG 知识库。内容面向普通健身、减脂、增肌、饮食记录、训练计划、用户长期记录分析等场景，适合被切分、向量化后用于问答增强。

## 建议目录结构
```text
knowledge_base/
  00_README_RAG使用说明.md
  01_能量平衡_BMR_TDEE_热量目标.md
  02_三大营养素_蛋白质_碳水_脂肪.md
  03_减脂策略_体重管理_平台期.md

  06_常见动作提示_安全与纠错.md

```

## RAG 检索建议
建议按 Markdown 标题和段落切分，chunk_size 可先设为 500–900 中文字符，chunk_overlap 设为 80–150 中文字符。健身问答建议检索 top_k=4～6；如果用户问题同时涉及饮食和训练，可以分别检索「饮食文档」与「训练文档」后合并上下文。

## 推荐回答上下文拼接顺序
1. 用户当前问题
2. 用户基础信息：性别、年龄、身高、体重、目标、训练频率
3. 最近身体数据：体重、体脂、腰围、瘦体重变化
4. 最近 7–14 天饮食记录：热量、蛋白质、碳水、脂肪、餐次
5. 最近 7–14 天训练记录：训练部位、动作、组数、次数、重量、主观感受
6. RAG 检索到的知识片段
7. 安全边界：非医疗建议，有疾病/疼痛/不适建议就医或咨询专业人员

## 适合测试的问题
- 我 175cm、102kg，想减脂，一周练 5 天，蛋白质应该怎么安排？
- 根据我最近 7 天的饮食记录，我哪里需要调整？
- 我今天练胸背，卧推 100kg 5x5，明天应该怎么练？
- 我这周体重没下降，是不是平台期？
- 深蹲时膝盖不舒服，可能有哪些动作问题？

## 注意
本知识库用于健康教育和健身建议，不用于诊断疾病、治疗伤病或替代医生/营养师/康复师建议。
